# Linux Post-Install Scripts  

## Usage

First, download the file to your computer. Open Terminal and cd into the appropriate location. Make the file executable:
```
sudo chmod +x *-postinstall-script.sh 
```

Run the file:
```
./*-postinstall-script.sh 
```
